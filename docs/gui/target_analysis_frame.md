# `gui/target_analysis_frame.py`

## الدور

إدخال **معلومات هدف** (اسم، نوع، OS، خدمات، إصدار، وصف) واستدعاء **`AIAnalyzer.analyze_target`** لعرض تحليل دفاعي وتوصيات مختبر آمن.

## الشكل

حقول نموذج عمودية أو شبكة؛ زر تحليل؛ منطقة نص لعرض ناتج Markdown من النموذج.

## البيانات

لا يخزّن بالضرورة في DB؛ الناتج عرض فوري من Groq.
