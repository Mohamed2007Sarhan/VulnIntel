# `lab/sandbox_workflow.py`

## الدور

توليد **سير عمل مختبر آمن** خطوة بخطوة لزوج (CVE, exploit) وهدف اختياري.

## الدالة الرئيسية

`generate_workflow(cve, exploit, target=None) -> Dict`

يُرجع هيكلًا يحتوي `phases`: شروط مسبقة، تحقق، خطوات ما بعد، مع نصوص توجيهية معزولة عن الإنتاج.

## الاستخدام

يمكن عرضه في إطار Lab أو التقارير عند الربط.
