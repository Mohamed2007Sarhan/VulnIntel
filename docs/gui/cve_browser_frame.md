# `gui/cve_browser_frame.py`

## الدور

**استعراض وبحث CVE** من SQLite: حقل بحث، قائمة نتائج، وتفاصيل CVE مختارة (الوصف، CVSS، المراجع، إلخ حسب التطبيق).

## السلوك النموذجي

- `refresh()` يعيد تحميل القائمة من `db.search_cves` أو ما يماثله.  
- `_select_cve` يُستدعى خارجيًا من Dashboard عند النقر على بطاقة.

## الشكل

منطقة بحث في الأعلى؛ قائمة قابلة للتمرير؛ لوحة تفاصيل لجهة CVE النشطة.
