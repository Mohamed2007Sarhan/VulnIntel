# `main.py`

## المسار في المستودع

`main.py` (جذر المشروع)

## الدور

نقطة **الدخول الوحيدة** لتشغيل تطبيق سطح المكتب VulnIntel.

## ماذا يفعل بالتحديد؟

1. يضيف جذر المشروع إلى `sys.path` حتى تعمل الاستيرادات (`gui.app`, `config`, …).  
2. يضبط **التسجيل** (`logging`): مستوى INFO، تنسيق مع الطابع الزمني، مخرجات إلى الـ stdout.  
3. يعرّف `main()` التي تستورد `VulnIntelApp` من `gui.app` وتنشئ النسخة وتشغّل `mainloop()` (حلقة Tkinter).  
4. أي استثناء غير متوقع يُسجَّل ويخرج بالرمز 1.

## من يستدعيه؟

المستخدم أو IDE يشغّل:

```bash
python main.py
```

## التبعيات

- `gui.app.VulnIntelApp`  
- لا يعتمد على قاعدة البيانات مباشرة؛ التهيئة داخل `VulnIntelApp.__init__`.
